============================================================
QUÉ DEBE IR EN CADA CARPETA
============================================================

/assets/brand/logo-arete.svg
Logo de Team ARETÉ / Programa ARETÉ.
Ideal: SVG blanco o versión premium simple.
Si solo tenés PNG, usar /assets/brand/logo-arete.png y actualizar el asset map.

/assets/brand/favicon.svg
Favicon de la web.
Puede ser logo simplificado.

/assets/hero/hero-video.mp4
Video principal actual del hero.
Debe estar optimizado para web.
Recomendado: mp4, H.264, peso bajo, muted, playsInline.

/assets/hero/hero-poster.webp
Imagen poster del video, visible antes de cargar/reproducir.
Puede ser una captura del video.

/assets/lucas/lucas-main.webp
Foto principal de Lucas para hero o bloque fuerte.
Debe verse premium/editorial.

/assets/lucas/lucas-about.webp
Foto de Lucas para sección “Quién es Lucas Casalins”.
Puede ser la imagen actual de Lucas en escenario/competición.

/assets/lucas/lucas-training.webp
Foto de Lucas entrenando o en contexto fitness premium.

/assets/testimonials/case-01.webp ... case-22.webp
Imágenes de casos de éxito / transformaciones.
Usar tantas como existan.
No es obligatorio tener exactamente 22, pero si el copy lista 22 casos, idealmente cargar 22 imágenes.

============================================================
SI FALTAN IMÁGENES
============================================================

Si no tenés alguna imagen:
- Mantener el item de copy.
- En LandingCopy.js dejar image: null.
- El componente debe tener fallback elegante: gradiente oscuro, número grande, textura sutil.
- No usar imágenes random de internet.

============================================================
URLS / CTAS
============================================================

CTA principal:
Usar href: "#booking" si el objetivo es bajar a agenda.

Si existe un link externo real de WhatsApp, GHL o calendario, reemplazarlo en LandingCopy.js:
cta: {
  label: "QUIERO EMPEZAR",
  href: "#booking"
}
